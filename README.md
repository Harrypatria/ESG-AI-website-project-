## 🌟 Overview

**PT AII (Agentic Industrial Intelligence)** is a high-end, production-grade web platform showcasing the frontier of industrial autonomy. It bridges the gap between complex AI research and operational excellence, focusing on three core pillars: **Agentic AI**, **Climate Impact (ESG)**, and **Industry Mastery**.

> *"The future of industry isn't just about automation. It's about agency — systems that perceive, reason, and act to optimize both profit and planet."*
> — **PT AII Leadership**

### Key Capabilities

| Capability | Description |
|---|---|
| 🤖 **Agentic AI** | Showcasing autonomous systems that move beyond fixed logic into reasoning-based execution. |
| 🌍 **Climate Impact** | Real-time visualization of ESG metrics and carbon reduction strategies through precision AI. |
| 🏭 **Industry Excellence** | Operational mastery through predictive maintenance and supply chain optimization. |
| 🧪 **Research Frontier** | Deep dives into the science of agentic systems and their future trajectories. |
| 🚀 **Interactive Demos** | Live trials of AI agents, providing a hands-on experience of autonomous intelligence. |

## 🏗️ Core Sections

The application is modularized into distinct sections, each representing a facet of PT AII's mission:

- **Hero**: The "Industrial Intelligence, Redefined" entrance with immersive animations.
- **Agentic AI**: Explaining the transition from automation to autonomy.
- **Climate Impact**: Interactive charts (Recharts) showing ESG performance and carbon reduction.
- **Industry Excellence**: Radar charts comparing traditional vs. AI-augmented operational metrics.
- **Research**: A grid-based showcase of the latest scientific breakthroughs in agentic AI.
- **Future**: A visionary look at the autonomous horizon of 2030 and beyond.
- **Demos**: A curated list of interactive AI agent trials with a sophisticated modal system.
- **Experts**: Highlighting the multi-disciplinary team driving the innovation.

## ⚙️ Setup & Installation

### Prerequisites

- Node.js v18+
- npm or yarn

### Local Development

```bash
# 1. Clone the repository
git clone <repository-url>
cd pt-aii-web

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Add your GEMINI_API_KEY if needed for agentic features

# 4. Start the development server
npm run dev
# → Open http://localhost:3000
```

### Build for Production

```bash
# Compile and optimize for production
npm run build

# The output will be in the /dist directory
```

## 🛠️ Development Guide

To maintain the high-end aesthetic and performance of the PT AII web platform, please follow these guidelines:

1. **Typography**: Use the `Inter` font family for general UI. Maintain a clean, sentence-case aesthetic for headings.
2. **Animations**: Use `motion` for component-level animations and `GSAP` for complex, scroll-triggered sequences.
3. **Styling**: Stick to Tailwind CSS 4.0 utility classes. Avoid inline styles. Use the `cn()` utility for conditional classes.
4. **Icons**: Always import from `lucide-react`.
5. **Responsiveness**: Design desktop-first but ensure mobile-first implementation using Tailwind's responsive prefixes (`sm:`, `md:`, `lg:`, `xl:`).
6. **Data Viz**: Use `Recharts` for all data-driven components. Ensure tooltips and axes are styled to match the dark/glassmorphic theme.
